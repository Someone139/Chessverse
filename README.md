# Chessverse

A fully playable web-based chess platform built with Next.js, React, chess.js, and react-chessboard.

---

## Live Demo
https://chessverse-tau.vercel.app

---

## Features

- Fully playable chess game
- Drag and drop piece movement
- Legal move validation (chess.js)
- Turn-based gameplay
- Move history tracking
- Check and checkmate detection
- Pawn promotion system

---

## Tech Stack

- Next.js (App Router)
- React
- chess.js
- react-chessboard
- Tailwind CSS

---

## Purpose

This project was built as a learning journey into:
- game logic
- React state management
- UI/UX design
- full-stack deployment with Vercel
